# Anti-Spasticity_Hand-Device
## Developing an Anti-Spasticity Hand Device Using Raspberry Pi 4, Air Pump, and Pressure Sensor
- Raspberry Pi 4 Model B
- BMP180 pressure sensor
- Air pump 12v
- Relay Module
- 3 buttons for mode selection
